🚀 Connect EasyPayDirect to ClickFunnels

Quick & Easy Setup Guide

Ready to connect your EasyPayDirect gateway to ClickFunnels? Here's how to do it. Takes about 10 minutes.

💡 Before You Start: Open both your EasyPayDirect merchant portal and ClickFunnels in separate tabs - makes it easier!

🔑 Step 1: Get Your Security Key from EasyPayDirect

1Log into your EasyPayDirect merchant portal
  • Go to your merchant portal (the link was in your welcome email)
  • Log in with your username and password

EasyPayDirect merchant portal login page

2Navigate to Security Keys
  • Click on "Gateway Options" in the menu
  • Then click "Security Keys"

Your Security Keys page will look like this

3Copy your Private Security Key
  • Look for the section called "Private Security Keys"
  • Find the key labeled "API" (NOT the "Cart" one)
  • Click the "Copy" button next to it
  • Save it somewhere safe (you'll need it in a minute!)
⚠️ Important: Make sure you copy the Private key (API), not the Public key. The Private key is what connects ClickFunnels to your gateway securely.

⚙️ Step 2: Add the Gateway in ClickFunnels

4Go to Payment Gateways in ClickFunnels
  • Log into your ClickFunnels account
  • Click your account name (top right corner)
  • Select "Account Settings"
  • Click "Payment Gateways" in the left menu

ClickFunnels Payment Gateway page

5Add New Gateway
  • Click the "+ Add New Payment Gateway" button
  • Select "Network Merchants Inc (NMI)" from the list
  • Click "Add Gateway"
Why NMI? EasyPayDirect uses the Network Merchants Inc (NMI) gateway format for ClickFunnels integration.
6Enter Your Credentials

Fill out the form with these values:

Gateway Name: Type EasyPayDirect (or whatever you want to call it)

API Login ID: Type api_key (literally those letters - not a placeholder!)

Transaction Key: Paste your Private Security Key from Step 3

Gateway Mode: Select Test Mode (for now)

MD5 Hash (if asked): Type gateway
⚠️ That "api_key" thing looks weird, right? Don't worry - you really do type the literal word "api_key" in the API Login ID field. It's how EasyPayDirect's system works. Your actual authentication happens through the Private Security Key.

Fill in your credentials like this

7Save It!
  • Double-check everything looks right
  • Click "Save"
  • You should see a success message

🎯 Step 3: Connect to Your Funnel

8Select Your Funnel
  • Go to your Funnels page
  • Click on the funnel you want to add payments to
  • Find your Order Form or Checkout page
  • Click "Edit Page"
9Assign the Payment Gateway
  • Click on your Order Form element
  • In the settings panel, find "Payment Gateway"
  • Select "EasyPayDirect" from the dropdown
  • Make sure your products and prices are set correctly
  • Click "Save" then "Publish"

Select EasyPayDirect in your order form settings

🧪 Step 4: Test It!

10Run a Test Transaction

Before going live, let's make sure everything works!

  1. Preview your funnel (click the Preview button)
  2. Go through your checkout page
  3. Use this test card:
Field Value
Card Number 4111 1111 1111 1111
Expiration Any future date (like 12/27)
CVV 123
Name Test Customer
Zip Code Any valid zip
  1. Complete the test order
  2. You should see a success/thank you page
  3. Check your EasyPayDirect portal - you should see the test transaction
✅ Success looks like: You get to a thank you page, receive a confirmation email, and see the transaction in your EasyPayDirect portal under Transactions.

🎉 Step 5: Go Live!

11Switch to Live Mode

Once your test works perfectly:

  1. Go back to ClickFunnels → Account Settings → Payment Gateways
  2. Click on your EasyPayDirect gateway
  3. Change from "Test Mode" to "Live Mode"
  4. Click "Save"
⚠️ Important: Only switch to Live Mode after your test transaction works perfectly!

🎊 You're All Set!

Your ClickFunnels store is now connected to EasyPayDirect and ready to accept payments!

What happens now:

  • Customers can pay with credit/debit cards
  • Transactions process securely through EasyPayDirect
  • Funds settle to your bank account
  • You can view all transactions in your merchant portal

❓ Need Help?

Common Questions

Q: It says "Invalid Credentials" - what's wrong?

A: Double-check that you typed api_key exactly in the API Login ID field, and that you pasted your Private Security Key (API) - not the Public key - in the Transaction Key field.

Q: The test card is being declined

A: Make sure your gateway is still in Test Mode. Also verify you're using 4111 1111 1111 1111 with a future expiration date.

Q: Which ClickFunnels plan do I need?

A: The Platinum plan or higher is recommended for full payment gateway features, but basic payment processing works on most plans.

Q: Can I use this for recurring/subscription payments?

A: Yes! ClickFunnels supports recurring billing with EasyPayDirect. Just set up your products as subscriptions in ClickFunnels.
